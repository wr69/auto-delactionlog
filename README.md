# auto-delactionlog

使用方法：创建变量 
 
1. 名字：GH_TOKEN 内容的写法：ghp_xxxxxxxxxx

2. 名字：LINK 内容的写法：

 - 备注|自己的公共库名字

 - 用回车键隔开

 - 例如: 当前库|wr69/auto-delactionlog
